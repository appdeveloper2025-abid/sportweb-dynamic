## рҹҗӣ TROUBLESHOOTING

### **Problem: Can't access website**
**Solution:**
- Check Apache is running (green in XAMPP)
- Use correct URL: `http://localhost/Static Sport web 2026/`

### **Problem: Database error**
**Solution:**
- Check MySQL is running (green in XAMPP)
- Verify database name: `kpk_sports`
- Re-import schema.sql

### **Problem: Can't login**
**Solution:**
- Use correct email: `admin@kpk.com`
- Password: `admin123`
- Check if admin user exists in database

### **Problem: Blank page**
**Solution:**
- Check if schema.sql was imported
- Verify all 5 tables exist
- Restart Apache
